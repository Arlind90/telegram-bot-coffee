Coffee Price Bot

This is a Telegram bot that sends daily coffee price updates at 9:00 AM.

Status: The bot is running! This page serves as a landing page only. All bot functionality is handled through serverless functions.

How it works

  • The bot communicates with users through Telegram
  • Updates are automatically sent at 9:00 AM daily
  • Uses Netlify Functions for serverless operation

To use the bot, search for it on Telegram and start a conversation!
